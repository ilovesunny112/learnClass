# js中的一些内置类
- 关于数据类型的
  + Number：每一个数字或者NaN都是他的一个实例
  + String：字符串类
  + Boolean：布尔值
  + Null
  + Undefined（浏览器屏蔽了我们操作null或者undefined这个类）
  + Object: 对象类，每一个对象数据类型都是他的实例
     + Array： 数组类
     + RegExp:  正则类
     + Date: 日期类
  + Function: 函数类，每一个函数都是它的一个实例
- 关于元素对象和元素集合
  + HTMLCollection: 元素集合类
      + getElementsByTagName/getElementsByClassName/querySelectorAll）
  + NodeList: 节点集合类
	  + getElementsByName
      + childNodes
