# 元字符讲解之开始结束转义
###  ^ $ \ 讲解
> var reg = /\d+/; //只要包含某某即可，这里说明含1到多个数字即可
> var str = 'zd201fsfd34'
> reg.test(str); //正则的匹配
> reg = /^\d+/ //中括号里面出现的^是非
> reg.test(reg)
> reg = /^\d+$/ //只能是1~9数字

>  // 转义字符
>  var reg = /^2.3$/ //-> /^2\.3$/ 
>
>   \\\ 两个斜杠正常解析
