# 闭包作用保存
> 函数执行形成一个私有作用域，函数执行完成，形成的这个栈内存一般情况下都会自动释放
> 其他情况：函数执行完成，当前私有作用域（栈内存）中的某一部分内容被栈内存以外的其他东西（变量/元素的事件）占用了，当前的栈内存就不能释放掉，也就形成了不销毁的私有作用域（里面的私有变量也不会销毁）

**`++i 和 i++区别`**
>  i++:去拿原有的值和其他的值进行运算，运算完成后再自身累加1
>  ++i: 先自身累加1，然后拿其累加晚的值和其他的值进行运算

### 练习题
```javascript
function fn() {
	var i = 1;
	return function(n) {
	 console.log(n + i++);
	}
}
var f = fn();
f(10);//11
fn()(10);//11
f(20)//22
fn()(20)//21
```
